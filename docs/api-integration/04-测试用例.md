# 前后端 API 联调 — 测试用例

> 环境：`VITE_USE_TEMU_BACKEND=true`（默认）、Java `:18080`、`npm run dev` `:5173`  
> 账号：见项目根 `租户账密.md`（如 Boss `HangZhouYiTuo` / `HangZhouYiTuo`）  
> 关联：`docs/permissions/04-回归测试用例.md`、`docs/temu-crawl/04-测试用例清单.md`

## 环境矩阵

| 模式 | 配置 | Java | 用途 |
|------|------|------|------|
| Backend | 默认或 `=true` | 必须 | 本清单主路径 |
| Demo | `VITE_USE_TEMU_BACKEND=false` | 可选 | REG-DEMO-* 回归 |

### 前置检查

```text
GET http://localhost:18080/api/auth/session  → 401（未登录）
GET http://localhost:5173                    → 前端可访问
```

## P0 — 契约与门禁

### API — 账户绑定

| ID | 场景 | 步骤 | 预期 | 结果 |
|----|------|------|------|------|
| API-P0-PA-01 | 绑定 snake_case | Boss Token POST `/api/platform-accounts/bind` body含 `store_name` | 200，`data.storeName` 有值 | |
| API-P0-PA-02 | 绑定 camelCase 失败 | 同上但仅 `storeName` | 400，店铺名称不能为空 | |
| API-P0-PA-03 | 列表 | GET `/api/platform-accounts` | 含刚绑定记录 | |
| API-P0-PA-04 | 删除 | DELETE `/api/platform-accounts/{id}` | 200，列表减少 | |
| API-P0-PA-05 | 员工无 Token | 无 Authorization GET list | 401 | |

### API — 组织与仓库

| ID | 场景 | 步骤 | 预期 | 结果 |
|----|------|------|------|------|
| API-P0-WS-01 | 新建分仓 | POST `/api/warehouse/sites` `sort_order:1` | 200，sortOrder 回显 | |
| API-P0-WS-02 | sortOrder 错误 | POST 仅 `sortOrder` 无 `sort_order` | 排序为 0 或后端默认（记录缺陷） | |
| API-P0-WF-01 | 新建仓管 | POST `/api/warehouse/members` `warehouse_ids:["站点id"]` | 200，warehouseIds 回显 | |
| API-P0-WF-02 | warehouseIds 错误 | 仅 camelCase `warehouseIds` | 仓库范围为空（记录缺陷） | |
| API-P0-EM-01 | 员工列表 | Boss GET `/api/tenant/members` | 200 数组 | |
| API-P0-EM-02 | 可分配菜单 | GET `/api/tenant/assignable-menus` | 含 `employee.warehouse` 等 | |
